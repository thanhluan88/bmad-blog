Practice Questions — PMBOK 8th Ed · Q591

Bài học Q591 về Develop Schedule (PMBOK 8, tr. 54) — vấn đề vendor / procurement

A construction project is underway. The project manager is concerned about discrepancies between the minor milestones outlined in the project management plan and the order in which the vendor is actually completing them. After discussing it with the vendor, the project manager realizes that certain contingencies have affected the way the vendor has organized their work. However, these contingencies should not prevent the vendor from delivering on schedule. What should the project manager do next?

2. Phân tích đáp án — Đáp án đúng: B

Grounding PMBOK 8

Dựa trên PMBOK 8 (Develop Schedule + Lead accountably, tr. 54): với vấn đề vendor / procurement, đáp án đúng là B — Quản lý vendor theo hợp đồng/procurement process — không bypass quy trình mua sắm. Tại sao chọn B?

Đáp án

B. Adjust the project plan to align with the vendor’s work approach and integrate it with the vendor’s perspective and work outputs.
