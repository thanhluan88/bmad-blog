Cheat sheet (cue → hành động)

IF budget forward · CPI/SPI · EAC · vendor · unpaid → Cập nhật forecast / EAC, hành động theo EVM; đóng contract khi admin-complete. IF estimation bias · vague policy · many changes delivered · lessons → Cập nhật OPA, template, rewrite policy — hành động được, không chỉ archive. IF MVP · mid-iteration · PO absent · backlog · sprint review → Remove impediment (escalate nếu cần); PO owns backlog; bảo vệ iteration — không hấp thụ mid-sprint. IF out of baseline · unauthorized work · new feature request · CCB → Impact assessment → Integrated Change Control (CR/CCB) — không absorb im lặng. IF sponsor confirm done · acceptance criteria · close check → Review scope / requirements / acceptance criteria trước khi claim success. IF risk materialized · new law · risk in reports · reserve → Tra risk register → implement planned response; identify/assess nếu mới. IF preferred channel · multicultural · virtual · language → Đổi kênh/phương pháp theo preference; giao thức nhạy văn hóa — không blast one-size. IF small low-risk · mixed certainty · predictive+agile → Tailor vừa đủ; hybrid OK; ước lượng adaptive khi bất định. IF Tuckman · storming · shouting · noisy workspace · ownership → Map Tuckman stage; EI/self-reg; PM sở hữu môi trường; facilitate ownership. IF interns · skill gap · training needs · onboarding → Gap analysis + develop/remediate (HR/mentor) — không blame/fire trước. IF steering review · RACI/RAM · decision authority · unclear duty → Làm rõ roles (RAM); chuẩn bị evidence governance; decision log. IF layoff · incomplete info · rumor · morale crash → Chia sẻ facts đã biết + tạo thảo luận mở — không im lặng hay hứa chắc chắn giả. IF resistant stakeholder · conflict interests · power/interest → Xem engagement plan; engage đúng chiến lược đã ghi — không freestyle. IF customer satisfaction · Kano · % value · long-term insight → Đo value đạt được (Kano/survey/benchmark) — tránh vanity metrics. IF other / mixed → open Q teach · don’t force bucket
