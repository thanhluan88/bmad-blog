A project is in its final delivery phase with a contractually committed deadline that cannot be moved without penalties. Several key team members unexpectedly request extended personal leave, which reduces available capacity and threatens to delay deliverables. The project manager informs stakeholders, who stress that missing the deadline would damage the company’s reputation and customer trust. What should the project manager do first?

Solution gốc (CSV — cột P)

C. Conduct an impact analysis and initiate a formal change request that balances project schedule, cost, and expectations. The project manager should perform an impact analysis and initiate a formal change request that balances the project schedule and cost. The project manager should not implement any changes without first understanding their impact. The other options are incorrect. Hiring temporary replacements would increase the budget and add to the schedule because the new resources would need to be trained before they could start working effectively. The project manager should request to dip into the contingency reserve only in the case of an emergency. Although this is an unplanned change in the amount of available resources, the project manager should explore other options first. Using reserves directly without justification could bypass governance and stakeholder agreement. Reviewing risk response strategies is a good idea, but that does not address the situation at hand. This answer assumes this situation was already documented in the risk register. Sudden personal leave of multiple key members often appears as an issue rather than as a previously identified risk. Although documenting it later is valid, it is not the immediate first step.
