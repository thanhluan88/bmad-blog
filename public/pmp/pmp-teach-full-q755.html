Answer

B. Compare estimated versus actual impacts on scope, cost, and schedule deliverables to quantify benefits.

Solution gốc (CSV — cột P)

B. Compare estimated versus actual impacts on scope, cost, and schedule deliverables to quantify benefits. The project manager should perform an impact analysis before any change is approved. An impact analysis looks at scope, cost, and schedule to determine whether the proposed change is feasible and beneficial. This aligns directly with the sponsor’s request to weigh benefits against measurable impacts. The other options are incorrect. An EV analysis measures project performance versus baseline (CPI, SPI). This is useful for tracking current health, but it does not evaluate a proposed change. A Monte Carlo simulation is a powerful forecasting tool, but it is too complex for the first step. The project manager should start with straightforward impact analysis before running advanced risk simulations. Cost analytics look only at the cost while ignoring the schedule impact and scope implications. The project manager should consider all three dimensions (scope, time, and cost), not cost alone.
