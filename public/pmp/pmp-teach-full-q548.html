Bài học Q548 về Develop Team (PMBOK 8, tr. 53) — trễ tiến độ / áp lực schedule

During a project quality review, the project manager displays the graph shown in the exhibit. The graph tracks the number of incidents over the last eight weeks. The graph shows a steady upward trend, with each week showing a steady increase in incidents. The project sponsor argues that no corrective action is required because the values remain below the thresholds. The quality manager warns that the trend indicates the process is degrading. The risk manager is concerned that the issue will escalate into schedule delays if not addressed. What should the project manager do? (Refer to the chart)

2. So sánh đáp án — Q548

A — Đúng Initiate a root cause analysis with the quality team because the trend pattern requires investigation.

Vấn đề lặp lại/quality — retrospective phân tích root cause và cải tiến hệ thống, không blame cá nhân.

B — Sai Monitor the trend because no thresholds are breached and the process remains formally within control.

"Monitor the trend because no thresholds are breached and the process remains forma…" không phù hợp trễ tiến độ / áp lực schedule — đáp án đúng tập trung facilitate retrospective / phân tích root cause.

C — Sai Ask the quality manager to suspend work and initiate an assessment of the potential severity of any risks.

yêu cầu bên liên quan/team thực hiện hành động không phù hợp trễ tiến độ / áp lực schedule — đáp án đúng tập trung facilitate retrospective / phân tích root cause.

D — Sai Postpone any responsive action until the next scheduled audit to confirm whether the trend continues.
