2. Lý luận PMBOK 8 (PMBOK reasoning)

  • Scenario: A project manager for a software development company faces a number of financial risks in their project.
  • PMBOK 8, p. 151 (Performance Domain: Governance, Manage Project Execution): In an environment where management activities are centralized, accountability (being answerable for an outcome) is usually assigned to one — which is why this PMBOK guidance applies directly to the stem and supports the correct PM action.
  • Therefore D is correct: 151: Manage Project Execution — aligns with this scenario.
