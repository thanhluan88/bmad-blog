Phân tích Câu 1050 — Bộ luyện PMBOK 8

A project team piloted a new dashboard for tracking earned value metrics that improved visibility and decision‑making. The dashboard was created within the project but is not yet part of the organization’s standard toolset. The project manager must ensure this innovation is captured for future use. What should the project manager do?

1. PMBOK 8 mapping

Framework

Domain: Stakeholders
Focus Area: Monitoring & Controlling
Process: Identify Stakeholders, Plan Stakeholder Engagement, Manage Stakeholder Engagement
Principle: Focus on value, Embed quality

2. So sánh đáp án

A — Sai Include the dashboard in the final project report to provide awareness for project stakeholders.

"Include the dashboard in the final project report to provide awareness for project…" không phù hợp mức độ uncertainty cao — đáp án đúng tập trung "Convert the pilot dashboard into a reusable template and add it to the organizatio…".

B — Đúng Convert the pilot dashboard into a reusable template and add it to the organizational repository.

Đáp án đúng theo PMBOK 8 và logic tình huống.

C — Sai Share the dashboard informally with other project managers across the organization.

Tập trung "thông báo / truyền đạt thông tin" không giải quyết mức độ uncertainty cao — đáp án đúng cần xử lý trực tiếp trọng tâm câu hỏi.

D — Sai Ask the project sponsor to approve the dashboard for organizational use in future initiatives.

Yêu cầu team tự xử lý — PM vẫn chịu trách nhiệm facilitate; đáp án đúng: "Convert the pilot dashboard into a reusable template and add it to the organizatio…".
