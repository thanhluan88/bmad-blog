During the planning stage of a project in a matrix organization, a project manager identifies the number and types of team members required to complete the work. Due to an ongoing change management initiative, functional managers report that the planned resources and personnel are not available for the project. Which two steps should the project manager take next? (Choose 2)

Why this answer

1. Reference solution

  • A. Investigate options for procuring or otherwise obtaining the required team members and resources from alternative sources.
  • Reevaluate the project objectives, review the planned activities, and recalculate the team member and resource needs based on current availability.

2. PMBOK 8 reasoning

  • Scenario: During the planning stage of a project in a matrix organization, a project manager identifies the number and types of team members required to complete the work.
  • PMBOK 8, p. 82 (Performance Domain: Resources, Acquire Resources): 2 Estimate Resources The key benefit of this process is that it identifies the type, quantity, and characteristics of resources required to complete the project. — which is why this PMBOK guidance applies directly to the stem and supports the correct PM action.
  • Therefore A is correct: Reevaluate the project objectives, review the planned activities, and recalculate the team member and resource needs based on current availability.

Answer

A,C

Solution gốc (CSV — cột P)

A, C A. Investigate options for procuring or otherwise obtaining the required team members and resources from alternative sources. C. Reevaluate the project objectives, review the planned activities, and recalculate the team member and resource needs based on current availability. Investigating procurement or alternative resourcing options directly is correct because it addresses the gap created by unavailable internal resources. Reevaluating objectives and recalculating resource needs is correct because it ensures the project plan reflects reality. The other answer choices are incorrect. Escalating to the sponsor and requesting a delay before exploring alternatives is incorrect because it is premature. The project manager should first investigate procurement options and reassess resource requirements before recommending a schedule change. Cancelling planning activities while waiting for the change initiative to end is incorrect because it is unnecessarily passive and may cause significant schedule delays. Submitting a formal conflict report to the PMO is incorrect because it is a reactive escalation step that should follow, not replace, the project manager's own analysis and resource exploration efforts.
