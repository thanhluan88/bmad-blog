# Sai:1 patterns — cue → action

| Pattern | Cue | Do | Don't |
|---|---|---|---|
| Artifact first | missing roles / SH list / unclear duties | RAM · stakeholder register · comms plan · charter | Freestyle meetings before artifacts |
| Change control | new feature / unauthorized work / reg mid-project | Impact analysis → formal CR/CCB | Absorb scope · keep unauthorized work |
| Compliance > pressure | deadline vs regulation / compliance knowledge gap | Keep compliance · expert judgment · backlog priority | Shortcut ethics/regs for speed |
| Clarify then act | SH doubts success / “not user-friendly” | Review acceptance/benefits · collaborate to clarify | Defend deliverable blindly · ignore concern |
| Agile intake & MVP | unplanned requests / MVP ready / weak PO | PO→backlog · ship MVP · negotiate PO capacity | Team accept mid-sprint · wait for perfect |
| Tailor / hybrid | small low-risk · evolving work · certification mid-design | Tailor governance · adaptive estimates · hybrid | One-size heavy plan |
| Risk cadence | planned response exists · new law · evolving risks | Implement planned response · register · reassess | Invent new path · ignore register |
| Culture / EI / environment | shouting · merger culture · noisy workspace | EI · cultural assess · PM owns environment | Ignore morale · treat as “not my job” |
