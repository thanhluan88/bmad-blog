Cheat sheet (cue → hành động)

IF budget forward · CPI/SPI · EAC · vendor · unpaid → Cập nhật forecast / EAC, hành động theo EVM; đóng contract khi admin-complete. IF MVP · mid-iteration · PO absent · backlog · sprint review → Remove impediment (escalate nếu cần); PO owns backlog; bảo vệ iteration — không hấp thụ mid-sprint. IF out of baseline · unauthorized work · new feature request · CCB → Impact assessment → Integrated Change Control (CR/CCB) — không absorb im lặng. IF estimation bias · vague policy · many changes delivered · lessons → Cập nhật OPA, template, rewrite policy — hành động được, không chỉ archive. IF risk materialized · new law · risk in reports · reserve → Tra risk register → implement planned response; identify/assess nếu mới. IF sponsor confirm done · acceptance criteria · close check → Review scope / requirements / acceptance criteria trước khi claim success. IF small low-risk · mixed certainty · predictive+agile → Tailor vừa đủ; hybrid OK; ước lượng adaptive khi bất định. IF preferred channel · multicultural · virtual · language → Đổi kênh/phương pháp theo preference; giao thức nhạy văn hóa — không blast one-size. IF Tuckman · storming · shouting · noisy workspace · ownership → Map Tuckman stage; EI/self-reg; PM sở hữu môi trường; facilitate ownership. IF resistant stakeholder · conflict interests · power/interest → Xem engagement plan; engage đúng chiến lược đã ghi — không freestyle. IF interns · skill gap · training needs · onboarding → Gap analysis + develop/remediate (HR/mentor) — không blame/fire trước. IF layoff · incomplete info · rumor · morale crash → Chia sẻ facts đã biết + tạo thảo luận mở — không im lặng hay hứa chắc chắn giả.
