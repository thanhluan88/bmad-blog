# Web — supplementary reasoning

Step 4b of [SKILL.md](SKILL.md). Fills **triad** lane 3 — [WHY.md#3-web](WHY.md#3-web).

## Query

From stem **conflict** + correct PM action (not bare domain):

```
{stem misconception or trap} + {correct option verb} + PMP exam
```

Examples:
- `SME agile teamwork slows quality misconception continuous improvement PMP`
- `undocumented risks status report vs risk register PMP exam`

Use `WebSearch`. Prefer: PMI, reputable exam prep, agile practice guides.

## Pick content

1. Confirms or sharpens why **this** key wins (exam logic, common trap)
2. Adds angle PMBOK lane lacks (e.g. agile misconception, FIRST action rule)
3. Stated in **prose** — 1–2 bullets

## Output

```json
{
  "whyWebBullets": [
    "Bullet — complete English sentence with reasoning."
  ],
  "whyWebSources": [
    "Site or article title (URL if found)"
  ]
}
```

## Done when

- 1–2 bullets; grammatical prose
- Source named per bullet or in `whyWebSources`
- Correct key agrees with solution + pmbok lanes
- Does not contradict column P

## Iterate

Weak or generic hit → refine query → search again (max 2 rounds).

Never fabricate statistics or fake citations.
